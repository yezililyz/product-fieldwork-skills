# Roadmap Prioritization

Covers whether to build at all (the ten-question investment discipline), the full Aha! Roadmaps workflow end-to-end (strategy → customer ideas → whiteboard brainstorm → scoring → roadmap → engineering handoff → KPI monitoring, with an AI assistant at every step), and interactive tools — a WSJF calculator, a BCG plotter, a SMART checker, a flow-efficiency gauge — for turning strategy into an actual portfolio of daily work.

## When to use this
- Building a prioritized roadmap from a raw backlog
- Running the full strategy-to-roadmap-to-engineering-handoff workflow
- Using a WSJF or BCG-style tool to score competing roadmap items
