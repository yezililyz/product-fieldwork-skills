# Building With AI Agents

Covers the practical mechanics of building agents that plan, use tools, remember, and act — not just chat.

## When to use this
- Architecting a multi-step agent or agentic workflow
- Deciding between a single LLM call vs. a full agent loop
- Giving an agent tool access (MCP) or memory/guardrails
- Automating a business process with AI end to end
