# AI Evals

Covers designing, grading, automating, and operationalizing AI evals across LLM and agent-based features.

## When to use this
- Setting up an eval suite for a new AI feature
- Deciding what "good" looks like for a model output
- Automating eval grading so it doesn't bottleneck shipping
