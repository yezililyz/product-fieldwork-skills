# AI Evals

A practical field guide for designing, grading, automating, and operationalizing AI evals across LLM and agent systems — the discipline that turns "seems to work" into something you can actually ship and trust.

## When to use this
- Setting up an eval suite for a new AI feature
- Deciding what "good" looks like for a model output before shipping it
- Automating eval grading so it doesn't bottleneck every release
- Debugging why an AI feature regressed after a model or prompt change
