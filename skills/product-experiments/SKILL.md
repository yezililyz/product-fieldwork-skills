# Product Experiments

A reusable guideline that walks one experiment's full arc — problem, analysis, model comparison, decision, safe rollout method, conclusion — with an interactive simulator for pressure-testing the specific trade-offs against the user's own assumptions.

## When to use this
- Designing one specific experiment's test/control setup
- Deciding what metric would actually prove the experiment worked
- Avoiding a false-positive result from an under-designed test
