# Evaluating Startup Ideas

Practical workflows, prompts, and specific places to source and validate startup ideas — using AI tools, Reddit, Upwork, and G2 — inspired by Sam Altman's take on the return of the "idea guy."

## When to use this
- Generating a shortlist of startup ideas worth exploring
- Deciding which of several ideas to pursue first
- Using AI tools and public forums (Reddit, G2, Upwork) to source and pressure-test ideas
