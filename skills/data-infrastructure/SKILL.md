# Data Infrastructure

The technical foundation layer: feature stores, data quality/lineage/security, integration architecture, and the APIs that expose it all.

## When to use this
- Designing a feature store or data pipeline for ML
- Explaining data lineage, quality, or governance to a non-technical stakeholder
- Designing or reviewing a public/internal API
- Deciding what data security/compliance a new system needs
